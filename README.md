🛒 Tech Shed – E-commerce Static Website

Tech Shed is a modern and responsive static e-commerce website built using HTML, CSS, JavaScript, and Bootstrap. It is designed to simulate an online tech store showcasing products like gadgets, accessories, and electronics with a clean and user-friendly interface.

✨ Features

🖥️ Fully responsive design for all devices

🛍️ Product listing with images, pricing, and descriptions
🛒 Add-to-cart interface (static/demo functionality)

📄 Product detail sections

🎯 Attractive banners and promotional sections

⚡ Smooth navigation and fast performance

🎨 Modern UI using Bootstrap components

🛠️ Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout design

JavaScript (ES6) – Interactive elements

Bootstrap 5 – Responsive framework and UI components
